## 概述

JAVA 通用编程技能，适用于 Spring Boot 项目。

---

## 核心规范

### 语言版本
- Java 17 或更高

### 框架
- Spring Boot 3.x
- Spring Data JPA
- Spring Security

### API 设计
- RESTful API 设计
- JSON 响应格式
- 统一异常处理

### 代码风格
- 单文件不超过 500 行
- 函数不超过 100 行
- 使用 Lombok 简化代码
