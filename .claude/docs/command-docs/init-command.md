# /init 命令说明

> **类别:** 项目初始化
> **更新:** 2025-03-22

---

## 功能

初始化项目 CLAUDE.md，创建项目级 AI 指导文档。

---

## 用法

```
/init
```

---

## 输出

生成 `CLAUDE.md` 文件，包含：
- 项目基础信息
- 技术栈说明
- 编码规范引用
- TODO 标记待补充内容

---

## 示例

```bash
# 在新项目目录中
/init

# 输出:
# ✅ CLAUDE.md 已创建
# 📝 请根据项目实际情况修改 TODO 部分
```

---

## 注意事项

- 该命令会覆盖现有 CLAUDE.md
- 建议在项目根目录执行
- 执行后需手动补充 TODO 内容
