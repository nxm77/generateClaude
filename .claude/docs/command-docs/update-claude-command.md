# /update-claude 命令说明

> **类别:** 文档更新
> **更新:** 2025-03-22

---

## 功能

增量更新项目 CLAUDE.md，自动生成代码相关内容。

---

## 用法

```
/update-claude
```

---

## 输出

更新 `CLAUDE.md` 中的自动生成部分：
- 函数列表 (通过代码分析)
- 数据操作列表
- API 端点列表
- 类/组件索引

---

## 示例

```bash
# 代码变更后更新文档
/update-claude

# 输出:
# ✅ CLAUDE.md 已更新
# 📊 新增 3 个函数
# 📊 更新 2 个 API 端点
```

---

## 注意事项

- 只更新自动生成部分
- 手动编写的内容不会被修改
- 建议在提交前执行
